### 🔹 WebSockets vs HTTP
| Feature | HTTP | WebSockets |
|--------|------|-----------|
| Connection | Stateless | Persistent |
| Communication | Request/Response | Full Duplex |
| Real-Time | ❌ No | ✅ Yes |
